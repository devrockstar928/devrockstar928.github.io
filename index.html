RPA Automation

I've automated Intelligenti Company's daily RPA report system and removed their boring/repetitive jobs.

The original workflow was that the reporters log into their RPA website and find the report that meets the criteria and download/upload it to s3 storage and visualizes the reports on the dashboard.

Of course, there is no problem if there isn't much report data but there was massive amount of data indeed.

I've used selenium and chrome driver(and ie driver) to automate this workflow.

The challenge was to avoid access restriction mechanism and bypass any kind of captcha blockers.

Crawlera proxy service and recaptcha v2 API did the trick.

And I've used d3.js to visualize the report data as a graph and diagram on the dashboard.
